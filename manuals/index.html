手册

本节包含关于如何安装、设置、配置和使用 Docker 产品的用户指南。

开源

开源开发和容器化技术。

Docker Build

在任何地方构建和交付任何应用程序。

Docker Engine

行业领先的容器运行时。

Docker Compose

定义和运行多容器应用程序。

Testcontainers

使用您喜欢的编程语言以编程方式运行容器。

Cagent

开源的多智能体解决方案，助您完成任务。

AI

所有 Docker AI 工具，尽在一个易于访问的位置。

Ask Gordon

借助您的个人 AI 助手，简化工作流程并充分利用 Docker 生态系统。

Docker Model Runner

查看和管理本地模型。

MCP Catalog and Toolkit

使用 MCP 服务器增强您的 AI 工作流程。

产品

面向创新团队的端到端开发者解决方案。

Docker Desktop

容器开发的指挥中心。

Docker Hardened Images

用于可信软件交付的安全、最小化镜像。

Docker Offload

在云中构建和运行容器。

Build Cloud

在云中更快地构建镜像。

Docker Hub

发现、共享和集成容器镜像。

Docker Scout

镜像分析和策略评估。

Docker Extensions

自定义您的 Docker Desktop 工作流程。

Testcontainers Cloud

在云中运行具有真实依赖项的集成测试。

平台

与 Docker 平台相关的文档，例如管理和订阅管理。

Administration

面向公司和组织的集中式可观察性。

Billing

管理账单和支付方式。

Accounts

管理您的 Docker 账户。

Security

面向管理员和开发人员的安全保障。

Subscription

Docker 产品的商业使用许可。

企业

面向 IT 管理员，提供关于如何大规模部署 Docker Desktop 的帮助，以及与安全相关功能的配置指导。
