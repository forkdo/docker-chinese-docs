# 构建镜像

## 关于本系列

学习如何构建适合生产的、精简高效的 Docker 镜像，这对于最小化开销并增强生产环境中的部署至关重要。

## 你将学到什么

- 理解镜像层
- 编写 Dockerfile
- 构建、标记和发布镜像
- 使用构建缓存
- 多阶段构建
