Docker Offload 是一项完全托管的服务，它让你能够使用你已经熟悉的 Docker 工具，将容器的构建和运行任务卸载到云端。它使开发者能够在虚拟桌面基础设施 (VDI) 环境或不支持嵌套虚拟化的系统中高效工作。

在以下主题中，了解 Docker Offload、如何设置它、将其用于你的工作流程，以及如何排查常见问题。
