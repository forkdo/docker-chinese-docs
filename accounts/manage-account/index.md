# 管理 Docker 账户

您可以使用 Docker Home 集中管理您的 Docker 账户，包括管理和安全设置。

> [!TIP]
>
> 如果您的账户关联了强制执行单点登录 (SSO) 的组织，您可能没有权限更新账户设置。
> 您必须联系管理员来更新您的设置。

## 更新账户信息

账户信息在您的**账户设置**页面上可见。您可以更新以下账户信息：

- 全名
- 公司
- 位置
- 网站
- Gravatar 电子邮件

要使用 Gravatar 添加或更新您的头像：

1. 创建一个 [Gravatar 账户](https://gravatar.com/)。
2. 创建您的头像。
3. 将您的 Gravatar 电子邮件添加到您的 Docker 账户设置中。

您的头像在 Docker 中更新可能需要一些时间。

## 更新电子邮件地址

要更新您的电子邮件地址：

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 在右上角选择您的头像，然后选择**账户设置**。
3. 选择**电子邮件**。
4. 输入您的新电子邮件地址和密码以确认更改。
5. 选择**发送验证电子邮件**。Docker 会向您的新电子邮件发送一个验证链接。

在您完成验证过程之前，您的新电子邮件地址将显示为未验证。您可以：

- 如果需要，重新发送验证电子邮件。
- 在验证前随时删除未验证的电子邮件地址。

要验证您的电子邮件，请打开您的电子邮件客户端并按照 Docker 验证电子邮件中的说明进行操作。

> [!NOTE]
>
> Docker 账户一次仅支持一个已验证的电子邮件地址，该地址用于账户通知和安全相关通信。您无法向您的账户添加多个已验证的电子邮件地址。

## 更改您的密码

您可以通过发起密码重置来更改密码。要更改密码：

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 在右上角选择您的头像，然后选择**账户设置**。
3. 选择**密码**，然后选择**重置密码**。
4. Docker 将向您发送一封密码重置电子邮件，其中包含重置密码的说明。

## 管理双重身份验证

要更新您的双重身份验证 (2FA) 设置：

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 在右上角选择您的头像，然后选择**账户设置**。
3. 选择**2FA**。

更多信息，请参阅[启用双重身份验证](../security/2fa/_index.md)。

## 管理个人访问令牌

要管理个人访问令牌：

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 在右上角选择您的头像，然后选择**账户设置**。
3. 选择**个人访问令牌**。

更多信息，请参阅[创建和管理访问令牌](../security/access-tokens.md)。

## 管理关联账户

您可以取消关联的 Google 或 GitHub 账户：

1. 登录您的 [Docker 账户](https://app.docker.com/login)。
2. 在右上角选择您的头像，然后选择**账户设置**。
3. 选择**关联账户**。
4. 在您关联的账户上选择**取消关联**。

要完全取消关联您的 Docker 账户，您还必须从 Google 或 GitHub 取消关联 Docker。更多信息请参阅 Google 或 GitHub 的文档：

- [管理您的 Google 账户与第三方之间的关联](https://support.google.com/accounts/answer/13533235?hl=en)
- [审查和撤销 GitHub Apps 的授权](https://docs.github.com/en/apps/using-github-apps/reviewing-and-revoking-authorization-of-github-apps)
