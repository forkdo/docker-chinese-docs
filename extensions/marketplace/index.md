# Marketplace 扩展

Extensions Marketplace 中提供两种类型的扩展：
- Docker 审核的扩展
- 自行发布的扩展

Docker 审核的扩展由 Docker Extensions 团队手动审核，以确保额外的信任和质量级别。它们在 Marketplace 中显示为 **Reviewed**（已审核）。

自行发布的扩展由扩展开发者自主发布，并经过自动化验证过程。它们在 Marketplace 中显示为 **Not reviewed**（未审核）。

## 安装扩展

> [!NOTE]
>
> 对于某些扩展，使用前需要创建单独的帐户。

安装扩展：

1. 打开 Docker Desktop。
2. 从 Docker Desktop 仪表板中，选择 **Extensions**（扩展）选项卡。
   扩展 Marketplace 将在 **Browse**（浏览）选项卡中打开。
3. 浏览可用的扩展。
   您可以按 **Recently added**（最近添加）、**Most installed**（安装最多）或按字母顺序对扩展列表进行排序。或者，使用 **Content**（内容）或 **Categories**（类别）下拉菜单，根据扩展是否已审核或按类别搜索扩展。
4. 选择一个扩展并选择 **Install**（安装）。

在此之后，您可以选择 **Open**（打开）以访问该扩展，或安装其他扩展。该扩展也会出现在左侧菜单和 **Manage**（管理）选项卡中。

## 更新扩展

您可以在 Docker Desktop 版本发布之外更新任何扩展。要将扩展更新到最新版本，请导航到 Docker Desktop 仪表板并选择 **Manage**（管理）选项卡。

**Manage**（管理）选项卡会显示您安装的所有扩展。如果扩展有新版本可用，它会显示一个 **Update**（更新）按钮。

## 卸载扩展

您可以随时卸载扩展。

> [!NOTE]
>
> 扩展使用的、存储在卷中的任何数据都必须手动删除。

1. 导航到 Docker Desktop 仪表板并选择 **Manage**（管理）选项卡。
   这将显示您已安装的扩展列表。
2. 选择要卸载的扩展右侧的省略号。
3. 选择 **Uninstall**（卸载）。
