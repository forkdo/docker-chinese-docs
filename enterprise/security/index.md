# 企业级安全

Docker 为管理员和开发者提供安全护栏。

如果您是管理员，您可以为您的开发者在所有 Docker 产品中强制登录，并通过增强容器隔离和注册表访问管理等 DevOps 安全控制来扩展、管理和保护您的 Docker Desktop 实例。
