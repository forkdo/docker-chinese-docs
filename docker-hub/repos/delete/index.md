# 删除仓库

> [!WARNING]
>
> 删除仓库会同时删除其中包含的所有镜像和构建设置。此操作无法撤销。

1. 登录到 [Docker Hub](https://hub.docker.com)。
2. 选择 **My Hub** > **Repositories**。

   您的仓库列表将显示出来。

3. 选择一个仓库。

   仓库的 **General** 页面将显示出来。

4. 选择 **Settings** 标签页。
5. 选择 **Delete repository**。
6. 输入您的仓库名称以确认。
7. 选择 **Delete Repository Forever**。
