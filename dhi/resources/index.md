## Docker Hub

Docker Hardened Images 可在 Docker Hub 上获取：

- [Docker Hardened Images 目录](https://dhi.io)：浏览并从官方目录拉取 Docker Hardened Images
- [Docker Hub MCP 服务器](https://hub.docker.com/mcp/server/dockerhub/overview)：MCP 服务器，用于列出您的组织中可用的 Docker Hardened Images (DHIs)

## GitHub 仓库和资源

Docker Hardened Images 仓库可在 [docker-hardened-images](https://github.com/docker-hardened-images) GitHub 组织中找到：

- [目录](https://github.com/docker-hardened-images/catalog)：DHI 定义文件和目录元数据
- [安全公告](https://github.com/docker-hardened-images/advisories)：随 DHIs 分发的 OSS 包的 CVE 安全公告
- [密钥环](https://github.com/docker-hardened-images/keyring)：公共签名密钥和验证工具
- [日志](https://github.com/docker-hardened-images/log)：Docker Hardened Images 的引用日志（标签 > 摘要）
- [讨论](https://github.com/orgs/docker-hardened-images/discussions)：社区论坛和产品讨论
