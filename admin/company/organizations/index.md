了解如何使用 Docker 管理控制台管理公司中的组织。

## 查看所有组织

1. 登录 [Docker Home](https://app.docker.com) 并选择您的公司。
2. 选择 **Admin Console**，然后选择 **Organizations**。

**Organizations** 视图会显示您公司下的所有组织。

## 向组织添加席位

如果您拥有 [自助服务](../../subscription/details.md#self-serve) 订阅，且没有待处理的订阅变更，则可以使用 Docker Home 添加席位。有关添加席位的更多信息，请参阅 [管理席位](/manuals/subscription/manage-seats.md#add-seats)。

如果您拥有销售协助订阅，则必须联系 Docker 支持或销售团队以添加席位。

## 向公司添加组织

要将组织添加到公司，请确保满足以下条件：

- 您是公司所有者。
- 您是要添加的组织的所有者。
- 该组织拥有 Docker Business 订阅。
- 公司下可存在的组织数量没有限制。

> [!IMPORTANT]
>
> 将组织添加到公司后，无法将其从公司中移除。

1. 登录 [Docker Home](https://app.docker.com)，然后从左上角的帐户下拉菜单中选择您的公司。
2. 选择 **Admin Console**，然后选择 **Organizations**。
3. 选择 **Add organization**。
4. 从下拉菜单中选择要添加的组织。
5. 选择 **Add organization** 进行确认。

## 管理组织

1. 登录 [Docker Home](https://app.docker.com)，然后从左上角的帐户下拉菜单中选择您的公司。
2. 选择 **Admin Console**，然后选择 **Organizations**。
3. 选择要管理的组织。

有关管理组织的更多详细信息，请参阅 [组织管理](../organization/_index.md)。
