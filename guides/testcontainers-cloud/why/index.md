# 为什么选择 Testcontainers Cloud？

Testcontainers Cloud 是一个功能强大的基于云的解决方案，旨在通过将容器管理转移到云端来优化 Testcontainers 的集成测试。它帮助开发人员和团队克服传统本地和基于 CI 测试的限制，确保环境一致性、更快的测试执行和可扩展的工作流程。无论您是 Testcontainers 的新手，还是希望增强现有设置，Testcontainers Cloud 都提供了一种无缝的方式来管理容器化测试，提高开发管道中的效率和可靠性。

Testcontainers Cloud 提供以下优势：

- **云端卸载：** 通过将容器管理转移到云端，释放本地资源，保持笔记本电脑的响应能力。
- **一致的测试环境：** 确保测试在隔离、可靠的环境中运行，减少从开发到 CI 各平台间的不一致性。
- **可扩展性：** 允许同时运行大量容器，不受本地或 CI 资源的限制。
- **更快的 CI/CD 管道：** 通过将容器卸载到多个按需云工作节点，并使用 Turbo-mode 功能，减少配置瓶颈并加快构建时间。

Testcontainers Cloud 通过将容器管理卸载到云端，确保环境一致性和更快的测试执行，从而减少资源压力，使其成为提高基于 Testcontainers 工作流程稳定性的重要工具。
