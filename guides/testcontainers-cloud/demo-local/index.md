本演示展示了通过 Docker 设置 Testcontainers Cloud 的过程，使其能够使用 Testcontainers Desktop 应用程序在您的本地开发环境中工作。在本演练结束时，您将成功启动并运行 Testcontainers Cloud by Docker，准备将容器管理从本地机器卸载到云端，以实现更高效的测试。

- 安装并配置 Testcontainers Cloud 和 CLI，以与您的本地开发环境无缝集成。
- 设置并配置 Testcontainers Desktop 应用程序，以便在本地测试期间监控和管理基于云的容器。
- 创建并使用 Testcontainers 运行集成测试，利用基于云的容器资源。
- 高效监控和管理容器，了解 Testcontainers Cloud 如何自动执行清理并确保一致的测试环境。
- 在 Testcontainers Cloud Dashboard 中查看用于监控和故障排除的选项。
