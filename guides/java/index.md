# Java 语言专属指南
Java 入门指南将教你如何使用 Docker 创建容器化的 Spring Boot 应用程序。在本模块中，你将学习如何：

- 使用 Maven 容器化并运行 Spring Boot 应用程序
- 设置本地开发环境，将数据库连接到容器、配置调试器，并使用 Compose Watch 实现实时重载
- 在容器内运行单元测试
- 使用 GitHub Actions 为你的应用程序配置 CI/CD 管道
- 将容器化的应用程序本地部署到 Kubernetes，以测试和调试你的部署

完成 Java 入门模块后，你应该能够基于本指南中提供的示例和说明来容器化你自己的 Java 应用程序。
