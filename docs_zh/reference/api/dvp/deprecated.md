本页面概述了 Docker Verified Publisher API 中已弃用的端点。

## 端点弃用策略

随着 Docker 的不断更新，有时需要移除现有端点或用新端点替换。在移除现有端点之前，文档中会将其标记为“已弃用”。一段时间后，该端点可能会被移除。

## 已弃用的端点

下表概述了当前已弃用端点的状态：

**已弃用 (Deprecated)**：端点被标记为“已弃用”，不应再使用。

该端点可能会在未来的版本中被移除、禁用或更改行为。

**已移除 (Removed)**：端点已被移除、禁用或隐藏。
