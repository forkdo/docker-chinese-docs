本文档概述了 Docker Verified Publisher API 中已弃用的端点。

## 端点弃用策略

随着 Docker 的变更，有时需要移除或替换现有的端点，并以更新的端点替代。在移除现有端点之前，会在文档中标记为“已弃用”。经过一段时间后，这些端点可能会被移除。

## 已弃用的端点

下表概述了当前已弃用端点的状态：

**已弃用**：端点已标记为“已弃用”，不应继续使用。

该端点可能在未来的版本中被移除、禁用或更改行为。

**已移除**：端点已被移除、禁用或隐藏。
