## docker desktop disable

禁用 Docker Desktop

### 简介

禁用 Docker Desktop。这将停止 Docker Desktop 服务并将其从启动项中移除。

### 语法

```shell
docker desktop disable [flags]
```

### 示例

禁用 Docker Desktop：

```shell
docker desktop disable
```

### 选项

| 选项 | 描述 |
| --- | --- |
| `-h, --help` | 显示 `docker desktop disable` 命令的帮助信息 |
