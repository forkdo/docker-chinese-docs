# docker desktop enable

启用 Docker Desktop

## 概要

```
docker desktop enable [OPTIONS]
```

## 描述

启用 Docker Desktop。

## 选项

| 选项 | 描述 |
| --- | --- |
| `-h, --help` | 显示使用信息 |

## 示例

启用 Docker Desktop：

```
$ docker desktop enable
Docker Desktop is now enabled.
```
