# docker desktop kubernetes images

列出 Kubernetes 集群中已缓存的镜像

## 摘要

```
docker desktop kubernetes images [OPTIONS]
```

## 描述

列出 Kubernetes 集群中已缓存的镜像

## 选项

| 选项 | 描述 |
| --- | --- |
| `--kubeconfig kubeconfig` | Kubernetes 配置文件的路径 |
| `-o`, `--output format` | 使用 Go 模板格式化输出 |
| `-q`, `--quiet` | 只显示镜像 ID |
