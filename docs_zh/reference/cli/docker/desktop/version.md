# docker desktop version

显示 Docker Desktop 版本

## 概要

```
docker desktop version [OPTIONS]
```

## 描述

显示 Docker Desktop 版本

## 选项

| 选项 | 描述 |
| --- | --- |
| `-f, --format` | 使用自定义格式打印版本信息 |
| `-h, --help` | 打印使用信息 |
