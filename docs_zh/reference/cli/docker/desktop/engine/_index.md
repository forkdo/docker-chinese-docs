# docker desktop engine

管理 Docker Desktop 引擎

## 概要

管理 Docker Desktop 引擎

```
docker desktop engine [flags]
docker desktop engine [command]
```

## 可选项

```
  -h, --help   显示 engine 命令的帮助信息
```

## 可用命令

| 命令 | 描述 |
|---|---|
| [docker desktop engine restart](./docker_desktop_engine_restart.md) | 重启 Docker Desktop 引擎 |
