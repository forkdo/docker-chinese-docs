## docker desktop restart

重启 Docker Desktop

### 摘要

```
docker desktop restart [flags]
```

### 选项

```
      --background   在后台重启 Docker Desktop
  -h, --help         显示 restart 命令的帮助信息
```

### 从父命令继承的选项

```
  -D, --debug   启用调试模式
```
