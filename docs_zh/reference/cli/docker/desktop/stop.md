# docker desktop stop

停止 Docker Desktop

## 概要

```
docker desktop stop [flags]
```

## 描述

停止 Docker Desktop。


## 选项

| 选项 | 描述 |
| --- | --- |
| `-h`, `--help` | 获取 `stop` 命令的帮助信息 |
