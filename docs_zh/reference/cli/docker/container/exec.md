> **介绍 Docker Debug**
>
> 要轻松获取到任何容器的调试 shell，请使用 `docker debug`。Docker Debug
> 是 `docker exec` 调试的替代方案。使用它，您可以无需修改即可进入任何容器或镜像的 shell，
> 即使是精简版的镜像也可以。此外，您还可以在可自定义的工具箱中携带您最喜欢的调试工具。
>
> 立即探索 [Docker Debug](../debug.md)。
