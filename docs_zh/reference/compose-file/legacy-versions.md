Compose 文件参考的旧版本已迁移至 [Compose 仓库的 V1 分支](https://github.com/docker/compose/tree/v1/docs)。这些版本已不再进行主动维护。

最新且推荐的 Compose 文件格式由 [Compose Specification](_index.md) 定义。该格式合并了 2.x 和 3.x 版本，并由 **Compose 1.27.0+** 实现。更多信息，请参阅 [Docker Compose 的历史和发展](/manuals/compose/intro/history.md)。
