Compose 文件参考的旧版本已移至 [Compose 仓库的 V1 分支](https://github.com/docker/compose/tree/v1/docs)。这些版本不再被积极维护。

Compose 文件格式的最新和推荐版本由 [Compose 规范](_index.md) 定义。此格式合并了 2.x 和 3.x 版本，并由 **Compose 1.27.0+** 实现。更多信息，请参阅 [Docker Compose 的历史与发展](/manuals/compose/intro/history.md)。
