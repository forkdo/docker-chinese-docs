有多种方式可以为 Docker Offload 提供反馈。

## 快速调查

最快速分享想法的方式是填写这个简短的
[Docker Offload 反馈调查](https://docker.qualtrics.com/jfe/form/SV_br8Ki4CCdqeIYl0)。只需一分钟即可完成，有助于 Docker 团队改善你的使用体验。

## 产品内反馈

在每个 Docker Desktop Dashboard 页面上，都有一个 **Give feedback**（提供反馈）链接。点击后会打开反馈表单，你可以直接与 Docker 团队分享建议。

## 在 GitHub 上报告错误或问题

要报告错误或问题，请访问：
- [GitHub 上的 Docker Desktop for Mac 问题](https://github.com/docker/for-mac/issues)
- [GitHub 上的 Docker Desktop for Windows 问题](https://github.com/docker/for-win/issues)
- [GitHub 上的 Docker Desktop for Linux 问题](https://github.com/docker/desktop-linux/issues)
