Docker Offload 是一项完全托管的服务，让您使用熟悉的 Docker 工具将容器的构建和运行卸载到云端。它使开发者能够在虚拟桌面基础架构（VDI）环境或不支持嵌套虚拟化的系统中高效工作。

在以下主题中，了解 Docker Offload 的相关信息，包括如何设置、在工作流中的使用方式，以及常见问题的排查方法。
