了解如何使用 Admin Console 更新您的组织信息。

## 更新组织信息

常规组织信息会显示在 Admin Console 中的组织登录页面。

这些信息包括：

 - 组织名称
 - 公司
 - 位置
 - 网站
 - Gravatar 邮箱：要为 Docker 账户添加头像，请创建 [Gravatar 账户](https://gravatar.com/) 并上传头像。接下来，将您的 Gravatar 邮箱添加到 Docker 账户设置中。您的头像可能需要一些时间才能在 Docker 中更新。

要编辑这些信息：

1. 登录 [Admin Console](https://app.docker.com/admin) 并从左上角的账户下拉菜单中选择您的组织。
1. 输入或更新组织的详细信息，然后选择 **保存**。
