了解如何使用 Admin Console 更新组织信息。

## 更新组织信息

组织的一般信息会显示在 Admin Console 中组织的主页上。

这些信息包括：

 - 组织名称
 - 公司
 - 位置
 - 网站
 - Gravatar 邮箱：要为你的 Docker 账户添加头像，请先创建一个 [Gravatar 账户](https://gravatar.com/) 并上传头像。然后，将你的 Gravatar 邮箱添加到 Docker 账户设置中。头像可能需要一些时间才能在 Docker 中更新。

要编辑这些信息：

1. 登录到 [Admin Console](https://app.docker.com/admin)，从左上角的账户下拉菜单中选择你的组织。
1. 输入或更新你组织的详细信息，然后选择 **保存**。
