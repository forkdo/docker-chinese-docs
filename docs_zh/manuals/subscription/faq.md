## 我可以将我的订阅从一个用户或组织账户转移到另一个账户吗？

订阅不能在账户或组织之间转移。

## 我可以暂停或延迟我的 Docker 订阅吗？

您无法暂停或延迟订阅，但可以降级您的订阅。如果订阅账单在到期日未支付，将从到期日起享有 15 天的宽限期。

## Docker 是否提供学术定价？

请联系 [Docker 销售团队](https://www.docker.com/company/contact) 了解学术定价选项的相关信息。

## 我如何为 Docker 内容做贡献？

Docker 提供两种内容贡献计划：

- [Docker 赞助开源计划 (DSOS)](../docker-hub/repos/manage/trusted-content/dsos-program.md)，面向开源项目
- [Docker 认证发布商 (DVP)](../docker-hub/repos/manage/trusted-content/dvp-program.md)，面向商业出版商

您也可以加入 [开发者预览计划](https://www.docker.com/community/get-involved/developer-preview/) 或注册抢先体验计划，参与研究并试用新功能。
