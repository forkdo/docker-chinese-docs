Docker Desktop 根据 [Docker 订阅服务协议](https://www.docker.com/legal/docker-subscription-service-agreement) 授予许可。当您下载并安装 Docker Desktop 时，系统会要求您同意这些条款。

Docker 订阅服务协议规定：

 - Docker Desktop 免费适用于：
    - 小型企业（员工人数少于 250 人且年收入低于 1000 万美元）
    - 个人使用
    - 教育用途
    - 非商业性开源项目
- Docker Desktop 在以下情况下需要付费订阅：
    - 大型组织的专业使用
    - 政府机构
    - 超出免费层级限制的商业用途
- 包含 Docker Desktop 的付费订阅：
    - Docker Pro、Team 和 Business 订阅

## 了解许可条款

有关这些条款如何影响您的组织的详细信息，请参阅：

- [订阅更新博客文章](https://www.docker.com/blog/updating-product-subscriptions/)
- [Docker 订阅常见问题](https://www.docker.com/pricing/faq)，了解这可能如何影响使用 Docker Desktop 的公司。

> [!NOTE]
>
> Docker 和 Moby 开源项目（例如 Docker Engine）的许可和分发条款没有改变。

Docker Desktop 是使用开源软件构建的。有关 Docker Desktop 中开源组件许可的详细信息，请选择鲸鱼菜单 > **关于 Docker Desktop** > **致谢**。

## 开源组件

Docker Desktop 分发了一些在 GNU 通用公共许可证下授权的组件。[可从此处下载这些组件的源代码](https://download.docker.com/opensource/License.tar.gz)。
