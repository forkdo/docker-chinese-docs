Docker 订阅为个人开发者、团队和企业提供了各种功能和优势。本文档说明如何为个人账户和组织设置订阅。

开始之前，请确保您已拥有 [Docker ID](../accounts/create-account.md)。如需了解各订阅层级包含的内容，请参阅 [Docker 订阅和功能](./details.md) 和 [Docker 定价](https://www.docker.com/pricing/)。

## 为个人账户设置 Docker 订阅

当您 [创建 Docker ID](../accounts/create-account.md) 时，系统会自动为您分配 Docker Personal 订阅。此订阅免费提供基本的 Docker 工具。如果您希望继续使用 Docker Personal，则无需执行其他操作。您可以立即开始使用 Docker Desktop、Docker Hub 和其他工具。

如果您需要更多功能，如 Docker Build Cloud 分钟数和更多 Docker Scout 仓库，请参阅 [升级您的订阅](./change.md#upgrade-your-subscription)。

## 为组织设置 Docker 订阅

您可以为新组织或现有组织订阅 Docker Team 或 Business 订阅。只有组织所有者才能管理该组织的账单。

1. 如果您还没有 Docker ID，请先 [创建 Docker ID](../accounts/create-account.md)。
1. [创建您的组织](../admin/organization/orgs.md) 或使用您拥有的现有组织。
1. 为组织选择并购买 Docker Team 或 Business 订阅。

如需升级现有组织的订阅，请参阅 [升级您的订阅](./change.md#upgrade-your-subscription)。
