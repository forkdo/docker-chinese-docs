Docker 订阅允许您根据需求增长扩展使用量。所有付费的 Docker 订阅都包含基础数量的 Docker Build Cloud 构建分钟数和 Testcontainers Cloud 运行时分钟数，您可以通过额外容量进行补充。
您可以扩展以下资源的使用量：

- Docker Build Cloud 构建分钟数
- Testcontainers Cloud 运行时分钟数

要了解您的使用模式，可以随时 [查看使用情况](../admin/organization/manage-products.md#view-docker-product-usage)。

> [!NOTE]
>
> 使用旧版 Docker 订阅的用户必须升级到新的 Docker 订阅才能使用扩展选项。旧版订阅适用于在 2024 年 12 月 10 日之前最后一次购买或续订的用户。详情请见 [关于升级 Docker 套餐的公告](https://www.docker.com/blog/november-2024-updated-plans-announcement/)。

## 使用注意事项

分钟数不会结转。基础订阅分钟数在每个计费周期开始时重置，不会累积。额外购买的分钟数将在您的订阅期结束时过期。

例如，使用年度 Docker Team 订阅（包含 500 分钟），如果您购买了额外的 500 分钟，只有这额外的 500 分钟会结转到您的年度续订前使用。

## 添加 Docker Build Cloud 构建分钟数

您可以通过 Docker Build Cloud 仪表板购买额外的构建分钟数：

1. 登录到 [Docker Home](https://app.docker.com/)，选择您的组织。
1. 选择 **Build Cloud**，然后选择 **Build minutes**。
1. 选择 **Add prepaid minutes**。
1. 选择您需要的额外分钟数量，然后选择 **Continue to payment**。
1. 输入您的付款信息和账单地址。
1. 确认订单并选择 **Pay**。

您的额外分钟数将立即显示在构建分钟数页面上。

## 添加 Docker Testcontainers Cloud 运行时分钟数

您可以通过两种方式添加 Testcontainers Cloud 运行时分钟数：

- [联系销售](https://www.docker.com/pricing/contact-sales/) 以折扣价格预购运行时分钟数
- 使用按需计费的无限运行时分钟数，在每个计费周期结束时结算

按需使用的价格高于预购容量的价格。为避免较高的按需费用，如果您预期使用量持续超过订阅包含的分钟数，建议预购额外分钟数。
