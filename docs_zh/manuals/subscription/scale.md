Docker 订阅允许您根据需求增长扩展使用量。所有付费 Docker 订阅都包含 Docker Build Cloud 构建分钟数和 Testcontainers Cloud 运行分钟数的基础额度，您可以通过额外容量进行补充。
您可以扩展以下使用量：

- Docker Build Cloud 构建分钟数
- Testcontainers Cloud 运行分钟数

要了解您的使用模式，请随时[查看您的使用量](../admin/organization/manage-products.md#view-docker-product-usage)。

> [!NOTE]
>
> 旧版 Docker 订阅用户必须升级到新版 Docker 订阅才能访问扩展选项。旧版订阅适用于在 2024 年 12 月 10 日之前最后一次购买或续订的用户。有关详细信息，请参阅[宣布升级 Docker 计划](https://www.docker.com/blog/november-2024-updated-plans-announcement/)。

## 使用注意事项

分钟数不会结转。基础订阅分钟数在每个计费周期重置，不会累积。额外购买的分钟数将在您的订阅周期结束时过期。

例如，使用年度 Docker Team 订阅（包含 500 分钟），如果您额外购买 500 分钟，则只有这额外的 500 分钟会延续到年度续订。

## 添加 Docker Build Cloud 构建分钟数

通过 Docker Build Cloud 仪表板购买额外的构建分钟数：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
1. 选择 **Build Cloud**，然后选择 **Build minutes**。
1. 选择 **Add prepaid minutes**。
1. 选择您的额外分钟数，然后选择 **Continue to payment**。
1. 输入您的付款详情和账单地址。
1. 查看您的订单并选择 **Pay**。

您的额外分钟数将立即显示在 Build minutes 页面上。

## 添加 Docker Testcontainers Cloud 运行分钟数

您可以通过两种方式添加 Testcontainers Cloud 运行分钟数：

- [联系销售](https://www.docker.com/pricing/contact-sales/)以折扣价预购运行分钟数
- 按需使用无限运行分钟数，每月周期结束时计费

按需使用按比预购容量更高的费率计费。如果您预计使用量会持续超过订阅包含的分钟数，为避免更高的按需费用，请预购额外分钟数。
