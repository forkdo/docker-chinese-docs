## 18.06.3-ce

2019-02-19

### Docker Engine 的安全修复
* 更改 `runc` 关键漏洞补丁的应用方式，以在 RPM 包中包含该修复。[docker/engine#156](https://github.com/docker/engine/pull/156)

## 18.06.2

2019-02-11

### Docker Engine 的安全修复
* 更新 `runc` 以解决一个关键漏洞，该漏洞允许特制的容器获得主机上的管理员权限。[CVE-2019-5736](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2019-5736)
* 使用 3.13 内核的 Ubuntu 14.04 客户端需要升级到受支持的 Ubuntu 4.x 内核

## 18.06.0-ce
2018-07-18

### 关于此版本的重要说明

- Docker 18.06 CE 将是最后一个具有 4 个月维护生命周期的版本。计划中的 Docker 18.09 CE 版本将支持 7 个月，Docker 19.03 CE 将是下一个版本。有关发布流程的更多详细信息，请参阅[此处](/get-started/get-docker.md)。

### 构建器

* 构建器：修复多阶段通配符复制的层泄漏。[moby/moby#37178](https://github.com/moby/moby/pull/37178)
