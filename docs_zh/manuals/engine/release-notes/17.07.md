## 17.07.0-ce
2017-08-29

### API 和客户端

* 在 config.json 中添加对代理配置的支持 [docker/cli#93](https://github.com/docker/cli/pull/93)
* 默认启用 pprof/debug 端点 [moby/moby#32453](https://github.com/moby/moby/pull/32453)
* 现在可以使用 `docker login` 命令的新 `--password-stdin` 标志通过 `STDIN` 传递密码 [docker/cli#271](https://github.com/docker/cli/pull/271)
+ 在 docker scale 中添加 `--detach` [docker/cli#243](https://github.com/docker/cli/pull/243)
* 防止 `docker logs --no-stream` 因容器不存在而挂起 [moby/moby#34004](https://github.com/moby/moby/pull/34004)
- 修复 `docker stack ps` 将错误打印到 `stdout` 而非 `stderr` 的问题 [docker/cli#298](https://github.com/docker/cli/pull/298)
* 修复在 `docker service create` 部署期间发生错误时进度条卡住的问题 [docker/cli#259](https://github.com/docker/cli/pull/259)
* 改进交互模式下的进度条显示 [docker/cli#260](https://github.com/docker/cli/pull/260) [docker/cli#237](https://github.com/docker/cli/pull/237)
* 如果使用 `docker login --password`，打印警告并推荐使用 `--password-stdin` [docker/cli#270](https://github.com/docker/cli/pull/270)
* 使 API 版本协商更加健壮 [moby/moby#33827](https://github.com/moby/moby/pull/33827)
* 在连接到早于 Docker 17.05 的守护进程时隐藏 `--detach` [docker/cli#219](https://github.com/docker/cli/pull/219)
+ 在 `GET /networks/(id or name)` 中添加 `scope` 过滤器 [moby/moby#33630](https://github.com/moby/moby/pull/33630)

### 构建器

* 实现长时运行的交互式会话并增量发送构建上下文 [moby/moby#32677](https://github.com/moby/moby/pull/32677) [docker/cli#231](https://github.com/docker/cli/pull/231) [moby/moby#33859](https://github.com/moby/moby/pull/33859)

- 修复 `.dockerignore` 条目前导 `/` 不匹配任何内容的问题 [moby/moby#32088](https://github.com/moby/moby/pull/32088)

### 日志

- 修复轮转日志文件的错误文件模式 [moby/moby#33926](https://github.com/moby/moby/pull/33926)
