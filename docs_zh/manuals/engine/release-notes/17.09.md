## 17.09.1-ce
2017-12-07

### 构建器（Builder）

- 修复共享父阶段上的配置泄漏问题 [moby/moby#33753](https://github.com/moby/moby/issues/33753)
- 仅在空续行上发出警告，注释行除外 [moby/moby#35004](https://github.com/moby/moby/pull/35004)

### 客户端（Client）

- 即使 Ping 失败也在 Client 上设置 API 版本 [docker/cli#546](https://github.com/docker/cli/pull/546)

### 网络（Networking）

- Overlay 修复临时 IP 复用问题 [docker/libnetwork#2016](https://github.com/docker/libnetwork/pull/2016)
- 修复 NetworkDB 中的 reapTime 逻辑，并处理可附加容器的 DNS 清理 [docker/libnetwork#2017](https://github.com/docker/libnetwork/pull/2017)
- 在链存在检查中禁用主机名查找 [docker/libnetwork#2019](https://github.com/docker/libnetwork/pull/2019)
- 修复代码规范问题 [docker/libnetwork#2020](https://github.com/docker/libnetwork/pull/2020)
- 恢复 FindNetwork 中的错误类型 [moby/moby#35634](https://github.com/moby/moby/pull/35634)

### 运行时（Runtime）

- 保护 `health monitor` Go 通道 [moby/moby#35482](https://github.com/moby/moby/pull/35482)
- 修复容器/执行状态泄漏 [moby/moby#35484](https://github.com/moby/moby/pull/35484)
- 将 /proc/scsi 添加到掩码路径（用于修复 [CVE-2017-16539](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-16539) 的补丁）[moby/moby/#35399](https://github.com/moby/moby/pull/35399)
- 更新 tar-split：修复可能导致 Docker 守护进程崩溃的内存耗尽问题 [moby/moby/#35424](https://github.com/moby/moby/pull/35424) 修复 [CVE-2017-14992](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-14992)
- 修复 P/Z HubPullSuite 测试 [moby/moby#34837](https://github.com/moby/moby/pull/34837)
+ Windows：添加对拉取时版本过滤的支持 [moby/moby#35090](https://github.com/moby/moby/pull/35090)
- Windows：停止对 Windows 清单列表按版本过滤 [moby/moby#35117](https://github.com/moby/moby/pull/35117)
- 在 chroot 归档中使用 rslave 而非 rprivate [moby/moby/#35217](https://github.com/moby/moby/pull/35217)
- 在卸载后移除容器 rootfs 挂载路径 [moby/moby#34573](https://github.com/moby/moby/pull/34573)
- 修复用户 /dev/shm 挂载的 tmpfs 大小设置 [moby/moby#35316](https://github.com/moby/moby/pull/35316)
- 设置 may_detach_mounts 时不要中止（改为记录错误）[moby/moby#35172](https://github.com/moby/moby/pull/35172)
- 修复协商 API 版本时的版本比较 [moby/moby#35008](https://github.com/moby/moby/pull/35008)

### Swarm 模式（Swarm mode）

* 增加发送快照时的 gRPC 请求超时时间 [docker/swarmkit#2404](https://github.com/docker/swarmkit/pull/2404)

- 添加尝试更改集群名称时的错误 [docker/swarmkit/#2454](https://github.com/docker/swarmkit/pull/2454)

- 提供自定义 gRPC 拨号器以覆盖默认代理拨号器 [docker/swarmkit/#2457](https://github.com/docker/swarmkit/pull/2457)

## 17.09.0-ce
2017-09-26

+ 将 ulimits 添加到不受支持的 compose 字段 [docker/cli#482](https://github.com/docker/cli/pull/482)

* 当格式中包含 `{{.Digest}}` 时显示镜像摘要 [docker/cli#439](https://github.com/docker/cli/pull/439)

- 修复 `docker history --format {{json .}}` 在 `--human=true` 时打印人类可读时间戳而非 ISO8601 [docker/cli#438](https://github.com/docker/cli/pull/438)

- 修复服务使用多个 secrets 创建时的冗余服务重启 [moby/moby#34746](https://github.com/moby/moby/issues/34746)
