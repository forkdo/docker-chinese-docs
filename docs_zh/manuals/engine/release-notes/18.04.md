## 18.04.0-ce
2018-04-10

### 构建器

- 修复构建器和客户端中的拼写错误。 [moby/moby#36424](https://github.com/moby/moby/pull/36424)

### 客户端

* 在版本命令中打印 Stack API 和 Kubernetes 版本。 [docker/cli#898](https://github.com/docker/cli/pull/898)
- 修复版本命令中的 Kubernetes 重复问题。 [docker/cli#953](https://github.com/docker/cli/pull/953)
* 在帮助中使用 HasAvailableFlags 而非 HasFlags 处理选项。 [docker/cli#959](https://github.com/docker/cli/pull/959)
+ 添加对堆栈部署中必需变量的支持。 [docker/cli#893](https://github.com/docker/cli/pull/893)
- 修复 docker stack services 命令的端口输出。 [docker/cli#943](https://github.com/docker/cli/pull/943)
* 废弃未加密存储。 [docker/cli#561](https://github.com/docker/cli/pull/561)
* 不为 ConfigFile 设置默认文件名。 [docker/cli#917](https://github.com/docker/cli/pull/917)
- 修复 compose 网络名称。 [docker/cli#941](https://github.com/docker/cli/pull/941)

### 日志记录

* 静默登录：使用凭据存储中的凭据进行登录。 [docker/cli#139](https://github.com/docker/cli/pull/139)
+ 添加对日志文件压缩性的支持。 [moby/moby#29932](https://github.com/moby/moby/pull/29932)
- 修复非阻塞日志模式下的空 LogPath。 [moby/moby#36272](https://github.com/moby/moby/pull/36272)

### 网络

- 防止显式删除 ingress 网络。 [moby/moby#36538](https://github.com/moby/moby/pull/36538)
