本文档介绍如何恢复您的 Docker 账户以及如何管理双因素身份验证的恢复代码。

## 生成新的恢复代码

如果您丢失了双因素身份验证恢复代码，但仍能访问 Docker Hub 账户，您可以生成新的恢复代码。

1. 使用您的用户名和密码登录 [Docker 账户](https://app.docker.com/login)。
1. 选择您的头像，然后从下拉菜单中选择 **账户设置**。
1. 选择 **双因素身份验证 (2FA)**。
1. 输入您的密码，然后选择 **确认**。
1. 选择 **生成新代码**。

这将生成一个新的代码。选择可见性图标以查看代码。请保存您的恢复代码并将其存储在安全的地方。

## 无法访问账户时的恢复方法

如果您同时丢失了双因素身份验证应用程序和恢复代码：

1. 使用您的用户名和密码登录 [Docker 账户](https://app.docker.com/login)。
1. 选择 **我丢失了我的身份验证设备** 和 **我丢失了我的恢复代码**。
1. 填写 [联系支持表单](https://hub.docker.com/support/contact/?category=2fa-lockout)。

您必须在联系支持表单中输入与您的 Docker ID 关联的主要电子邮件地址，以获取恢复说明。
