双因素认证（2FA）通过在登录时除了密码外还需要一个唯一的验证码，为你的 Docker 账户增加了一层重要的安全保护。即使你的密码被盗，也能防止未经授权的访问。

当你启用双因素认证时，Docker 会为你的账户提供一个唯一的恢复代码。请将此代码安全保存，因为如果你无法访问验证器应用，可以使用它来恢复你的账户。

## 主要优势

双因素认证显著提升了你的账户安全性：

- 防止密码泄露：即使你的密码被盗或泄露，攻击者在没有你的第二个因素的情况下也无法访问你的账户。
- 安全的 CLI 访问：启用 2FA 后，Docker CLI 认证是必需的，确保自动化工具使用个人访问令牌而非密码。
- 合规要求：许多组织要求对访问开发和生产资源启用 2FA。
- 安心保障：了解你的 Docker 仓库、镜像和账户设置受到行业标准安全实践的保护。

## 前置条件

在启用双因素认证之前，你需要：

- 一部安装了基于时间的一次性密码（TOTP）验证器应用的智能手机或设备
- 访问你的 Docker 账户密码的权限

## 启用双因素认证

要为你的 Docker 账户启用 2FA：

1. 登录到你的 [Docker 账户](https://app.docker.com/login)。
1. 点击你的头像，然后从下拉菜单中选择 **Account settings**（账户设置）。
1. 选择 **2FA**。
1. 输入你的账户密码，然后选择 **Confirm**（确认）。
1. 保存你的恢复代码并将其存储在安全的地方。如果你无法访问验证器应用，可以使用恢复代码来恢复你的账户。
1. 使用 TOTP 移动应用扫描二维码或输入文本代码。
1. 将验证器应用生成的六位数字代码输入到文本框中。
1. 选择 **Enable 2FA**（启用 2FA）。

双因素认证现在已在你的账户上激活。每次登录时，你都需要输入验证器应用生成的安全代码。

## 禁用双因素认证

> [!WARNING]
>
> 禁用双因素认证会降低你的 Docker 账户的安全性。

1. 登录到你的 [Docker 账户](https://app.docker.com/login)。
2. 点击你的头像，然后从下拉菜单中选择 **Account settings**（账户设置）。
3. 选择 **2FA**。
4. 输入你的密码，然后选择 **Confirm**（确认）。
5. 选择 **Disable 2FA**（禁用 2FA）。
