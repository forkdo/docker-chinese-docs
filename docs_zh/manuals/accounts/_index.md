本节涵盖个人 Docker 账户和 Docker ID。不包括组织、公司或管理员角色。

Docker 账户用于：
- 创建 Docker ID
- 访问 Docker 产品和服务，如 Docker Hub 和 Docker Desktop
- 接收组织邀请
- 管理您的个人设置和安全功能
