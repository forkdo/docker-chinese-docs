### 什么是 Docker ID？

Docker ID 是您 Docker 账户的用户名，用于访问 Docker 产品。要创建 Docker ID，您需要以下其中一项：

- 一个电子邮件地址
- 一个社交账户
- 一个 GitHub 账户

您的 Docker ID 长度必须在 4 到 30 个字符之间，且只能包含数字和小写字母。不能使用任何特殊字符或空格。

更多信息，请参阅 [创建 Docker ID](/accounts/create-account/)。

### 我可以更改我的 Docker ID 吗？

不可以。一旦创建，您就无法更改 Docker ID。如果您需要不同的 Docker ID，必须创建一个具有新 Docker ID 的新 Docker 账户。

Docker ID 在停用后无法被重复使用。

### 如果我的 Docker ID 已被占用怎么办？

所有 Docker ID 都遵循先到先得的原则，拥有美国商标的公司除外。

如果您对某个命名空间拥有商标，[Docker 支持](https://hub.docker.com/support/contact/) 可以帮您取回该 Docker ID。

### 什么是组织名称或命名空间？

组织名称，有时也称为组织命名空间或组织 ID，是 Docker 组织的唯一标识符。组织名称不能与现有的 Docker ID 相同。
