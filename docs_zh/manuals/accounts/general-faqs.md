### 什么是 Docker ID？

Docker ID 是您 Docker 帐户的用户名，可让您访问 Docker 产品。要创建 Docker ID，您需要以下其中一项：

- 一个电子邮件地址
- 一个社交帐户
- 一个 GitHub 帐户

您的 Docker ID 必须在 4 到 30 个字符之间，并且只能包含数字和小写字母。您不能使用任何特殊字符或空格。

更多信息，请参阅 [创建 Docker ID](/accounts/create-account/)。

### 我可以更改我的 Docker ID 吗？

不可以。Docker ID 一旦创建便无法更改。如果您需要不同的 Docker ID，则必须使用新的 Docker ID 创建一个新的 Docker 帐户。

Docker ID 在停用后无法重复使用。

### 如果我的 Docker ID 被占用了怎么办？

所有 Docker ID 都是先到先得，但拥有用户名美国商标的公司除外。

如果您拥有您命名空间的商标，[Docker 支持](https://hub.docker.com/support/contact/) 可以为您找回该 Docker ID。

### 什么是组织名称或命名空间？

组织名称，有时称为组织命名空间或组织 ID，是 Docker 组织的唯一标识符。组织名称不能与现有的 Docker ID 相同。
