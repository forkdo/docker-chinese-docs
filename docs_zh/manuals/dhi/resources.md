## Docker Hub

Docker Hardened Images 可在 Docker Hub 上获取：

- [Docker Hardened Images Catalog](https://dhi.io): 浏览并从官方目录拉取 Docker
  Hardened Images
- [Docker Hub MCP Server](https://hub.docker.com/mcp/server/dockerhub/overview):
  用于列出组织中可用的 Docker Hardened Images (DHI) 的 MCP 服务器

## GitHub 仓库和资源

Docker Hardened Images 仓库可在
[docker-hardened-images](https://github.com/docker-hardened-images) GitHub
组织中找到：

- [Catalog](https://github.com/docker-hardened-images/catalog): DHI 定义
  文件和目录元数据
- [Advisories](https://github.com/docker-hardened-images/advisories): 随 DHI
  分发的开源软件包的 CVE 公告
- [Keyring](https://github.com/docker-hardened-images/keyring): 公钥签名
  和验证工具
- [Log](https://github.com/docker-hardened-images/log): Docker Hardened Images 的引用日志（tag > digest）
- [Discussions](https://github.com/orgs/docker-hardened-images/discussions):
  社区论坛和产品讨论
