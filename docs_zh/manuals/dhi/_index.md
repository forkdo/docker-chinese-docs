Docker Hardened Images (DHI) 是由 Docker 维护的最小化、安全且可用于生产的容器基镜像和应用镜像。DHI 旨在减少漏洞并简化合规性，可以轻松集成到您现有的基于 Docker 的工作流中，几乎无需重新配置工具。

DHI 提供两个层级：**DHI Free** 以免费方式提供核心安全功能，而 **DHI Enterprise** 则为有高级需求的组织添加了 SLA 支持、合规变体、定制化和扩展生命周期支持。
