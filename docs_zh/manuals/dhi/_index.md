Docker Hardened Images（DHI）是由 Docker 维护的精简、安全且可用于生产的容器基础镜像和应用镜像。DHI 旨在减少漏洞并简化合规性，可轻松集成到您现有的基于 Docker 的工作流中，几乎无需重新配置工具。

DHI 提供两个版本：**DHI Free** 提供核心安全功能，免费使用；而 **DHI Enterprise** 则增加了 SLA 支持、合规变体、自定义功能和扩展生命周期支持，适用于有高级需求的企业。
