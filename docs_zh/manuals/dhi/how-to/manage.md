在 Docker Hub 的 **Manage** 屏幕上，您可以管理组织中已镜像的 Docker Hardened Image (DHI) 仓库、已镜像的 DHI chart 仓库以及自定义项。

## 管理已镜像的 Docker Hardened Image 仓库

要管理已镜像的 DHI 仓库：

1.  前往 [Docker Hub](https://hub.docker.com) 并登录。
2.  选择 **My Hub**。
3.  在命名空间下拉菜单中，选择您的组织。
4.  选择 **Hardened Images** > **Manage**。
5.  选择 **Mirrored Images**。
6.  在您要管理的仓库的最右侧列中，选择菜单图标。

   在这里，您可以：

   - **Customize**：基于源仓库创建自定义镜像。
   - **Stop mirroring**：停止镜像 DHI 仓库。

## 管理自定义的 Docker Hardened Image 仓库

要管理自定义的 DHI 仓库：

1.  前往 [Docker Hub](https://hub.docker.com) 并登录。
2.  选择 **My Hub**。
3.  在命名空间下拉菜单中，选择您的组织。
4.  选择 **Hardened Images** > **Manage**。
5.  选择 **Customizations**。

   在此页面上，您可以查看自定义的 DHI 仓库。

6.  在您要管理的仓库的最右侧列中，选择菜单图标。

   在这里，您可以：

   - **Edit**：编辑自定义镜像。
   - **Create new**：基于源仓库创建新的自定义镜像。
   - **Delete**：删除自定义镜像。
