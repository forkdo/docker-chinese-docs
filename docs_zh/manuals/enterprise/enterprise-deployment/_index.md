Docker Desktop 支持可扩展的部署选项，专为企业 IT 环境定制。无论您是将 Docker 推广到数百个开发人员工作站，还是通过 Intune 或 Jamf 等 MDM 解决方案强制实施一致的配置，本节都提供了以安全、可重复的方式安装、配置和管理 Docker Desktop 所需的一切。了解如何使用 MSI 和 PKG 安装程序、配置默认设置、控制更新，以及确保符合组织策略——涵盖 Windows、macOS 和 Linux 系统。
