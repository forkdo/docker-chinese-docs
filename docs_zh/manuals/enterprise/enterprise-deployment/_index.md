Docker Desktop 支持专为企业级 IT 环境设计的可扩展部署选项。无论您是要在数百个开发人员工作站上部署 Docker，还是通过 Intune 或 Jamf 等 MDM 解决方案强制实施一致的配置，本节都提供了以安全、可重复的方式安装、配置和管理 Docker Desktop 所需的全部内容。了解如何使用 MSI 和 PKG 安装程序、配置默认设置、控制更新，并确保符合您组织的策略——适用于 Windows、macOS 和 Linux 系统。
