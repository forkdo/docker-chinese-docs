您可以更新个人账户或组织的账单信息。更新账单信息后，这些更改将应用于未来的账单发票。您为账单账户提供的电子邮件地址是 Docker 发送所有发票和其他账单相关通信的地方。

> [!NOTE]
>
> 已存在的发票（无论是否已支付）无法更新。
> 更改仅适用于未来的发票。

## 管理账单信息

### 个人账户

{{< tabs >}}
{{< tab name="Docker 订阅" >}}

要更新您的账单信息：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
1. 选择 **Billing（账单）**。
1. 在左侧导航中选择 **Billing information（账单信息）**。
1. 在您的账单信息卡片上，选择 **Change（更改）**。
1. 更新您的账单联系人和账单地址信息。
1. 可选。要添加或更新增值税号（VAT ID），请勾选 **I'm purchasing as a business（作为企业购买）** 复选框并输入您的税号。

    > [!IMPORTANT]
    >
    > 您的增值税号必须包含国家前缀。例如，如果您输入的是德国的增值税号，则应输入 `DE123456789`。

1. 选择 **Update（更新）**。

{{< /tab >}}
{{< tab name="旧版 Docker 订阅" >}}

要更新您的账单信息：

1. 登录 [Docker Hub](https://hub.docker.com)。
1. 选择您的组织，然后选择 **Billing（账单）**。
1. 选择 **Billing Address（账单地址）** 并输入您的更新后的账单信息。
1. 可选。要添加或更新增值税号（VAT ID），请输入您的 **Tax ID/VAT（税号/增值税号）**。

    > [!IMPORTANT]
    >
    > 您的增值税号必须包含国家前缀。例如，如果您输入的是德国的增值税号，则应输入 `DE123456789`。

1. 选择 **Submit（提交）**。

{{< /tab >}}
{{< /tabs >}}

## 更新您的账单邮箱地址

Docker 会发送以下与账单相关的电子邮件：

- 确认邮件（新订阅、已支付发票）
- 通知（卡片失败、卡片过期）
- 提醒（订阅续订）

您可以随时更新接收账单发票的邮箱地址。

### 个人账户

{{< tabs >}}
{{< tab name="Docker 订阅" >}}

要更新您的账单邮箱地址：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
1. 选择 **Billing（账单）**。
1. 在左侧导航中选择 **Billing information（账单信息）**。
1. 在您的账单信息卡片上，选择 **Change（更改）**。
1. 更新您的账单联系人信息并选择 **Update（更新）**。

{{< /tab >}}
{{< tab name="旧版 Docker 订阅" >}}

要更新您的账单邮箱地址：

1. 登录 [Docker Hub](https://hub.docker.com)。
1. 选择您的组织，然后选择 **Billing（账单）**。
1. 选择 **Billing Address（账单地址）**。
1. 在 **Billing contact（账单联系人）** 部分更新邮箱地址。
1. 选择 **Submit（提交）**。

{{< /tab >}}
{{< /tabs >}}

### 组织

{{< tabs >}}
{{< tab name="Docker 订阅" >}}

要更新您的账单邮箱地址：

1. 登录 [Docker Home](https://app.docker.com/) 并选择您的组织。
1. 选择 **Billing（账单）**。
1. 在左侧导航中选择 **Billing information（账单信息）**。

{{< /tab >}}
{{< tab name="旧版 Docker 订阅" >}}

1. 选择 **Submit（提交）**。
