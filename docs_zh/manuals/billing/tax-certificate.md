如果您是美国的客户且免征销售税，您可以向 Docker 支持部门提交有效的税务豁免证明。

如果您是全球客户且需缴纳增值税（VAT），请确保在更新账单资料时，在 VAT 编号前包含您的国家前缀。

## 前置条件

在提交证明之前：

- 客户名称必须与证明上的名称一致。
- 证明必须列出 Docker Inc. 作为销售方或供应商，并填写所有相关字段。
- 证明必须有签名、日期，且未过期。
- 您必须包含要应用该证明的所有账户的 Docker ID 或命名空间。

> [!IMPORTANT]
>
> 如果适用，您可以在多个命名空间中使用同一份证明。

## 注册税务证明

1. [提交 Docker 支持工单](https://hub.docker.com/support/contact?topic=Billing&subtopic=Tax%20information) 以启动注册税务证明的流程。
1. 在支持工单的 **主题（Subject）** 字段中输入 **Tax certificate**。
1. 在 **详情（Details）** 字段中输入 **Submitting a tax certificate**。
1. 系统将显示如何提交税务证明的说明。
1. 填写支持表单中的所有必填字段。
1. 在文件上传部分，通过拖放文件或将文件拖入上传区域，或选择 **Browse files** 来添加税务证明。
1. 选择 **Submit**。

Docker 支持团队将在需要额外信息时与您联系。一旦您的税务豁免状态应用到您的账户，您将收到一封来自 Docker 的电子邮件确认。
