如果您是美国客户且免征销售税，您可以向 Docker 支持团队提交有效的免税证明。

如果您是需要缴纳增值税的全球客户，请在更新计费资料时确保包含您的
[增值税号](/billing/history/#include-your-vat-number-on-your-invoice) 及国家前缀。

{{% include "tax-compliance.md" %}}

## 先决条件

提交证明前，请确认：

- 客户名称必须与证明上的名称一致。
- 证明必须将 Docker Inc. 列为卖家或供应商，并填写所有相关字段。
- 证明必须已签署、注明日期且未过期。
- 您必须提供所有要应用该证明的账户的 Docker ID 或命名空间。

> [!重要]
>
> 如果适用，您可以将同一份证明用于多个命名空间。

## 注册税务证明

1. [提交 Docker 支持工单](https://hub.docker.com/support/contact?topic=Billing&subtopic=Tax%20information) 以启动注册税务证明的流程。
2. 在支持工单的 **主题** 中输入 **Tax certificate**。
3. 在 **详情** 字段中，输入 **Submitting a tax certificate**。
4. 系统将显示如何提交税务证明的说明。
5. 填写支持表单上的所有必填字段。
6. 在文件上传部分，通过拖放文件或选择 **浏览文件** 来添加税务证明。
7. 选择 **提交**。

如果需要任何额外信息，Docker 的支持团队将与您联系。一旦您的免税状态应用于您的账户，您将收到 Docker 发送的电子邮件确认。
