每个仓库都可以包含描述、概述和分类，以帮助用户理解其用途和使用方法。添加清晰的仓库信息可确保他人能够找到您的镜像并有效使用。

您只能修改未归档仓库的仓库信息。如果仓库已归档，则必须先取消归档才能修改信息。详情请参阅 [取消归档仓库](../archive.md#unarchive-a-repository)。

## 仓库描述

描述会显示在使用 `docker search` 命令时的搜索结果中，以及 Docker Hub 上的搜索结果中。

请考虑以下仓库描述最佳实践。

- 总结用途。以简洁明确的方式说明镜像的用途。明确指出它是用于特定应用程序、工具或平台，还是具有特定用例。
- 突出关键特性或优势。简要提及主要优势或使其与众不同的独特特性。例如高性能、易于使用、轻量级构建，或与特定框架或操作系统的兼容性。
- 包含相关关键词。使用用户可能搜索的关键词以提高可见性，例如技术栈、用例或环境。
- 保持简洁。描述最多可包含 100 个字符。描述应控制在一到两句话内，确保在搜索结果中易于阅读。用户应能快速理解镜像的价值。
- 聚焦目标受众。考虑您的目标受众（开发者、系统管理员等），并直接针对他们的需求编写描述。

遵循这些实践可帮助描述在搜索结果中更具吸引力和有效性，从而为您的仓库带来更多相关流量。

### 添加或更新仓库描述

1. 登录 [Docker Hub](https://hub.docker.com)。

2. 选择 **My Hub** > **Repositories**。

   您的仓库列表将显示。

3. 选择一个仓库。

   仓库的 **General** 页面将显示。

4. 选择描述字段下方的铅笔图标。

5. 指定描述。

   描述最多可包含 100 个字符。

6. 选择 **Update**。

## 仓库概述

概述描述了您的镜像用途以及如何运行它。当仓库至少有一个镜像时，概述会显示在仓库的公开视图中。如果启用了自动构建，概述将在每次成功构建后从源代码仓库的 `README.md` 文件同步。

请考虑以下仓库概述最佳实践。

- 描述镜像内容、提供的功能以及使用原因。可包括使用示例或项目背后的团队。
- 说明如何开始使用该镜像运行容器。您可以包含在 Dockerfile 中使用镜像的最小示例。
- 列出关键镜像变体和标签，以及变体的用例。
- 链接到文档或支持网站、社区或邮件列表以获取更多资源。
- 提供镜像维护者的联系信息。
- 包含镜像的许可证以及需要时查找更多详细信息的位置。

### 添加或更新仓库概述

1. 登录 [Docker Hub](https://hub.docker.com)。

2. 选择 **My Hub** > **Repositories**。

   您的仓库列表将显示。

3. 选择一个仓库。

   仓库的 **General** 页面将显示。

4. 在 **Repository overview** 下，选择 **Edit** 或 **Add overview**。

   将显示 **Write** 和 **Preview** 选项卡。

5. 在 **Write** 下指定您的仓库概述。

   您可以使用基本 Markdown 并使用 **Preview** 选项卡预览格式。

6. 选择 **Update**。

## 仓库分类

您可以为 Docker Hub 仓库标记分类，表示镜像的主要预期用例。这使得用户能够更轻松地查找和浏览他们感兴趣问题领域的相关内容。

### 自动生成的分类

> [!NOTE]
>
> 自动生成的分类仅适用于 Docker 验证发布商和 Docker 赞助开源计划参与者。

对于在 Docker Hub 分类功能推出之前创建的仓库，分类已使用 OpenAI 基于仓库标题和描述自动生成并应用。

作为已自动分类仓库的所有者，如果您认为分类不准确，可以手动编辑分类。请参阅 [管理仓库分类](#manage-categories-for-a-repository)。

自动生成分类是一次性努力，旨在帮助为功能存在之前创建的仓库播种分类。新仓库不会自动分配分类。

### 管理仓库分类

您可以为仓库标记最多三个分类。

要编辑仓库的分类：

1. 登录 [Docker Hub](https://hub.docker.com)。
2. 选择 **My Hub** > **Repositories**。

   您的仓库列表将显示。

3. 选择一个仓库。

   仓库的 **General** 页面将显示。

4. 选择描述字段下方的铅笔图标。
5. 选择要应用的分类。
6. 选择 **Update**。

如果您缺少某个分类，请使用
[反馈链接](https://docker.qualtrics.com/jfe/form/SV_03CrMyAkCWVylKu)
告诉我们您希望看到哪些分类。
