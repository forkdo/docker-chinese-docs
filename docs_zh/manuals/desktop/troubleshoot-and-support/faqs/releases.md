### 新版本发布的频率如何？

新版本大约每月发布一次，除非遇到需要更早发布的紧急修复。

**软件更新**标签页中的**自动检查更新**设置默认为开启状态。这意味着当有新版本可用时，你会在 Docker 菜单中收到通知，Docker Desktop 仪表板上也会显示通知徽章。

你也可以通过勾选**始终下载更新**复选框，让 Docker Desktop 在后台自动下载新版本更新。

有时新版本会分阶段逐步推出，可能需要几天时间。因此，如果你愿意等待，更新很快就会出现。或者，你也可以在 Docker 菜单中选择**检查更新**来立即获取最新版本。

### 如何确保组织中的所有用户都使用相同版本？

这由你的 IT 管理员通过终端管理软件进行管理。
