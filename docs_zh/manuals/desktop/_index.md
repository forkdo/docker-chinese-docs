Docker Desktop 是一款适用于 Mac、Linux 或 Windows 环境的一键安装应用程序，可帮助你构建、共享和运行容器化应用及微服务。

它提供了一个直观的图形用户界面（GUI），让你能够直接从本地机器管理容器、应用和镜像。

Docker Desktop 简化了复杂的配置流程，节省了设置时间，让你专注于编写代码。它自动处理端口映射、文件系统问题和其他默认设置，并定期提供错误修复和安全更新。

Docker Desktop 与你偏好的开发工具和语言无缝集成，并通过 Docker Hub 提供对大量可信镜像和模板的访问。这使得团队能够加速开发、自动化构建、启用 CI/CD 工作流，并通过共享仓库实现安全协作。

## 核心功能

* 能够将任何应用容器化并共享到任意云平台，支持多种语言和框架。
* 快速安装和配置完整的 Docker 开发环境。
* 包含最新版本的 Kubernetes。
* 在 Windows 上，可在 Linux 和 Windows 容器之间切换，以构建应用。
* 基于原生 Windows Hyper-V 虚拟化的快速可靠性能。
* 在 Windows 机器上通过 WSL 2 原生支持 Linux 工作。
* 支持代码和数据的卷挂载，包括文件变更通知和轻松访问本地网络上的运行容器。
