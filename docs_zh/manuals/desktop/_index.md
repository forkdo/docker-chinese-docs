Docker Desktop 是一款适用于 Mac、Linux 或 Windows 环境的应用程序，只需点击一下即可完成安装，让您可以构建、共享和运行容器化应用程序及微服务。

它提供了一个直观的图形用户界面 (GUI)，让您可以直接从本地机器管理容器、应用程序和镜像。

Docker Desktop 减少了复杂设置所花费的时间，让您能够专注于编写代码。它会处理端口映射、文件系统问题以及其他默认设置，并定期通过错误修复和安全更新进行升级。

Docker Desktop 与您首选的开发工具和语言集成，并通过 Docker Hub 让您访问庞大的可信镜像和模板生态系统。这使团队能够加速开发、实现构建自动化、启用 CI/CD 工作流，并通过共享存储库进行安全协作。

## 主要功能

* 能够以多种语言和框架将任何应用程序容器化并共享到任何云平台。
* 快速安装和设置完整的 Docker 开发环境。
* 包含最新版本的 Kubernetes。
* 在 Windows 上，能够在 Linux 和 Windows 容器之间切换以构建应用程序。
* 借助原生 Windows Hyper-V 虚拟化技术，实现快速可靠的性能。
* 能够在 Windows 机器上通过 WSL 2 原生地在 Linux 上工作。
* 用于代码和数据的卷挂载，包括文件更改通知以及轻松访问 localhost 网络上的运行容器。
