Docker 扩展允许你在 Docker Desktop 中使用第三方工具，以扩展其功能。

你可以无缝地将你喜爱的开发工具连接到应用程序开发和部署工作流中。通过增强调试、测试、安全和网络功能来扩展 Docker Desktop，并使用扩展 [SDK](extensions-sdk/_index.md) 创建自定义附加组件。

任何人都可以使用 Docker 扩展，且安装数量无限制。

![扩展市场](/assets/images/extensions.webp)

## 有哪些可用的扩展？

目前提供由合作伙伴、社区构建的扩展以及 Docker 官方构建的扩展。

你可以在 [Docker Hub](https://hub.docker.com/search?q=&type=extension) 或 Docker Desktop 内的扩展市场中浏览可用扩展列表。
