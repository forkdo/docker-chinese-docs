Docker 扩展让您可以在 Docker Desktop 中使用第三方工具来扩展其功能。

您可以将您喜爱的开发工具无缝连接到您的应用程序开发和部署工作流中。通过调试、测试、安全和网络功能来增强 Docker Desktop，并使用扩展 [SDK](extensions-sdk/_index.md) 创建自定义附加组件。

任何人都可以使用 Docker 扩展，并且您可以安装的扩展数量没有限制。

![扩展市场](/assets/images/extensions.webp)

## 有哪些可用的扩展？

扩展包括合作伙伴和社区构建的扩展以及 Docker 构建的扩展。
您可以在 [Docker Hub](https://hub.docker.com/search?q=&type=extension) 或 Docker Desktop 内的扩展市场中探索可用的扩展列表。
