Extensions Marketplace 中有两种类型的扩展：
- Docker 审核的扩展
- 自主发布的扩展

Docker 审核的扩展由 Docker Extensions 团队进行人工审核，以确保额外的信任度和质量。它们在 Marketplace 中显示为 **已审核**。

自主发布的扩展由扩展开发者自主发布，并经过自动化验证流程。它们在 Marketplace 中显示为 **未审核**。

## 安装扩展

> [!NOTE]
>
> 对于某些扩展，在使用前需要单独创建账户。

要安装扩展：

1. 打开 Docker Desktop。
2. 从 Docker Desktop 仪表板中，选择 **扩展** 选项卡。
   扩展市场在 **浏览** 选项卡中打开。
3. 浏览可用的扩展。
   您可以按 **最近添加**、**安装最多** 或按字母顺序对扩展列表进行排序。或者，使用 **内容** 或 **类别** 下拉菜单，根据扩展是否已审核或按类别搜索扩展。
4. 选择一个扩展并选择 **安装**。

安装完成后，您可以选择 **打开** 来访问扩展，或安装其他扩展。该扩展也会出现在左侧菜单和 **管理** 选项卡中。

## 更新扩展

您可以在 Docker Desktop 版本之外随时更新任何扩展。要将扩展更新到最新版本，请导航到 Docker Desktop 仪表板并选择 **管理** 选项卡。

**管理** 选项卡会显示您已安装的所有扩展。如果某个扩展有新版本可用，它会显示一个 **更新** 按钮。

## 卸载扩展

您可以随时卸载扩展。

> [!NOTE]
>
> 扩展使用的、存储在卷中的任何数据必须手动删除。

1. 导航到 Docker Desktop 仪表板并选择 **管理** 选项卡。
   这会显示您已安装的扩展列表。
2. 选择要卸载的扩展右侧的省略号。
3. 选择 **卸载**。
