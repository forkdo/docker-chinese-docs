Extensions CLI 是一个用于管理 Docker 扩展的开发工具。支持的操作包括安装、列出、删除和验证扩展。

- `docker extension enable` 启用 Docker 扩展。
- `docker extension dev` 用于扩展开发的命令。
- `docker extension disable` 禁用 Docker 扩展。
- `docker extension init` 创建一个新的 Docker 扩展。
- `docker extension install` 使用指定的镜像安装 Docker 扩展。
- `docker extension ls` 列出已安装的 Docker 扩展。
- `docker extension rm` 删除一个 Docker 扩展。
- `docker extension update` 删除并重新安装一个 Docker 扩展。
- `docker extension validate` 根据 JSON 模式验证扩展元数据文件。
