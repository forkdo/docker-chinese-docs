## 示例

### 基本表格

| 权限级别                                                               | 访问权限                                                      |
| :--------------------------------------------------------------------- | :------------------------------------------------------------ |
| **粗体** 或 _斜体_ 在表格单元格内。下一个单元格故意留空。               |                                                               |
|                                                                        | 前一个单元格为空。单色文本中的 `--flag`。                       |
| 读取                                                                   | 拉取                                                          |
| 读写                                                                   | 拉取、推送                                                      |
| 管理员                                                                 | 上述所有权限，外加更新描述、创建和删除                           |

### 功能支持表格

| 平台       | x86_64 / amd64 |
| :--------- | :------------: |
| Ubuntu     |       ✅       |
| Debian     |       ✅       |
| Fedora     |                |
| Arch (btw) |       ✅       |

| 权限级别                                                               | 访问权限                                                      |
