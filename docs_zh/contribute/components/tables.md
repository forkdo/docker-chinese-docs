## 示例

### 基础表格

| 权限级别                                                                 | 访问权限                                                      |
| :----------------------------------------------------------------------- | :------------------------------------------------------------ |
| **粗体** 或 _斜体_ 在表格单元格内。下一个单元格特意留空。 |                                                               |
|                                                                          | 上一个单元格为空。`--flag` 为等宽文本。              |
| 读取                                                                     | 拉取                                                          |
| 读取/写入                                                               | 拉取、推送                                                    |
| 管理员                                                                    | 以上所有，外加更新描述、创建和删除 |

### 功能支持表格

| 平台   | x86_64 / amd64 |
| :--------- | :------------: |
| Ubuntu     |       ✅       |
| Debian     |       ✅       |
| Fedora     |                |
| Arch (btw) |       ✅       |

### 基础表格

```md
| 权限级别                                                                 | 访问权限                                                      |
| :----------------------------------------------------------------------- | :------------------------------------------------------------ |
| **粗体** 或 _斜体_ 在表格单元格内。下一个单元格特意留空。 |                                                               |

```

源文件中单元格的对齐方式并不重要。结尾的管道字符是可选的（除非最后一个单元格需要留空）。
