## 示例

使用破折号 (`-`) 或星号 (`*`) 表示项目符号列表。

- 项目符号列表项 1
- 项目符号列表项 2
- 项目符号列表项 3

1.  编号列表项 1。句号和第一个字母之间有两个空格，有助于对齐。

2.  编号列表项 2。让我们在其中添加一个注释。

    > [!NOTE]: 我们做到了！

3.  编号列表项 3，其中包含一个代码块。在代码块之前需要一个空行。

    ```bash
    $ docker run hello-world
    ```

4.  编号列表项 4，其中包含一个项目符号列表，项目符号列表中又包含一个编号列表。

    - 子项 1
    - 子项 2

      1.  子子项 1
      2.  子子项 2，其中包含一个表格，因为我们喜欢狂欢！缩进非常重要。

          | 标题 1 | 标题 2 |
          | ------ | ------ |
          | 项目 1 | 项目 2 |
          | 项目 3 | 项目 4 |
