## 示例

使用短横线 (`-`) 或星号 (`*`) 表示项目符号。

- 项目符号列表项 1
- 项目符号列表项 2
- 项目符号列表项 3

1.  编号列表项 1。句号与首字母之间保留两个空格
    有助于对齐。

2.  编号列表项 2。我们在这里添加一个提示。

    > [!NOTE]: 我们做到了！

3.  编号列表项 3，包含一个代码块。在代码块之前需要空一行。

    ```bash
    $ docker run hello-world
    ```

4.  编号列表项 4，包含一个项目符号列表，其内部又嵌套一个编号列表。

    - 子项 1
    - 子项 2

      1.  子子项 1
      2.  子子项 2，内部包含一个表格，因为我们喜欢复杂结构！
          缩进非常重要。

          | 表头 1 | 表头 2 |
          | -------- | -------- |
          | 项目 1  | 项目 2  |
          | 项目 3  | 项目 4  |
