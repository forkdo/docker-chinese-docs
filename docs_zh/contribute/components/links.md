## 示例

[外部链接](https://docker.com) 和 [内部链接](links.md) 都在同一标签页中打开。

请使用相对链接，并基于源文件名。

#### 指向自动生成内容的链接

当链接到自动生成页面（如 CLI 参考内容）中的标题 ID 时，编辑器将无法帮助你解析锚点名称。这是因为这些页面在构建时生成，你的编辑器或 LSP 无法预先知晓它们。

## 语法

```md
[外部链接](https://docker.com)
[内部链接](links.md)
```
