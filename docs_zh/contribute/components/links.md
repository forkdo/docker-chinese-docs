## 示例

[外部链接](https://docker.com) 和 [内部链接](links.md) 都在同一个标签页中打开。

请使用相对链接，基于源文件名。

#### 指向自动生成内容的链接

当你链接到自动生成页面中的标题 ID（例如 CLI 参考内容）时，无法从编辑器中获得锚点名称解析的帮助。这是因为这些页面是在构建时生成的，你的编辑器或 LSP 无法提前知道它们。

## 语法

```md
[外部链接](https://docker.com)
[内部链接](links.md)
```
