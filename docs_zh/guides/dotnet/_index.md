.NET 入门指南将教你如何使用 Docker 创建容器化的 .NET 应用。通过本指南，你将学会：

- 容器化并运行 .NET 应用
- 使用容器设置本地开发环境来开发 .NET 应用
- 使用容器运行 .NET 应用的测试
- 使用 GitHub Actions 为容器化的 .NET 应用配置 CI/CD 管道
- 将你的容器化应用本地部署到 Kubernetes 以测试和调试你的部署

完成 .NET 入门模块后，你应该能够根据本指南中提供的示例和说明，容器化你自己的 .NET 应用。

首先从容器化一个现有的 .NET 应用开始。
