Java 快速入门指南将教你如何使用 Docker 容器化 Spring Boot 应用程序。在本模块中，你将学习如何：

- 使用 Maven 容器化并运行 Spring Boot 应用
- 搭建本地开发环境，连接数据库到容器，配置调试器，并使用 Compose Watch 实现热重载
- 在容器内运行单元测试
- 使用 GitHub Actions 为你的应用配置 CI/CD 流水线
- 将容器化应用本地部署到 Kubernetes，以测试和调试你的部署

完成 Java 快速入门模块后，你应该能够根据本指南提供的示例和说明，容器化你自己的 Java 应用程序。

开始容器化你的第一个 Java 应用吧。
