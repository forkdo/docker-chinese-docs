PHP 语言专属指南将教你如何使用 Docker 创建容器化的 PHP 应用。在本指南中，你将学习如何：

- 容器化并运行 PHP 应用
- 设置使用容器开发 PHP 应用的本地环境
- 在容器内运行 PHP 应用的测试
- 使用 GitHub Actions 为容器化的 PHP 应用配置 CI/CD 流水线
- 将容器化的应用本地部署到 Kubernetes 以测试和调试部署

完成 PHP 语言专属指南后，你应该能够基于本指南提供的示例和说明来容器化你自己的 PHP 应用。
