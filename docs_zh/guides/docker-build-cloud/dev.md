借助 Docker Build Cloud，您可以轻松地将构建工作负载从本地机器转移到云端，从而帮助您实现更快的构建速度，尤其是在进行多平台构建时。

在本演示中，您将了解：

- 如何在本地设置构建器
- 如何将 Docker Build Cloud 与 Docker Compose 结合使用
- 镜像缓存如何为团队中的其他成员加速构建
