使用 Docker Build Cloud，您可以轻松地将构建工作负载从本地机器转移到云端，
从而帮助您实现更快的构建速度，特别是对于多平台构建。

在本演示中，您将看到：

- 如何在本地设置构建器
- 如何使用 Docker Build Cloud 和 Docker Compose
- 镜像缓存如何加速团队其他成员的构建
