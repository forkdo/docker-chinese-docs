Docker Build Cloud 可以显著减少你的 CI 构建运行所需的时间，为你节省时间和金钱。

由于构建在远程执行，你的 CI 运行器仍然可以使用 Docker 工具 CLI，而无需提升权限，这使得你的构建默认更加安全。

在本演示中，你将看到：

- 如何将 Docker Build Cloud 集成到各种 CI 平台
- 如何在 GitHub Actions 中使用 Docker Build Cloud 构建多架构镜像
- 使用 Docker Build Cloud 的工作流与本地运行的工作流之间的速度差异
- 如何在 GitLab Pipeline 中使用 Docker Build Cloud
