## 关于本系列

在本指南系列中，你将通过实践获得 Docker 的使用经验，从在本地机器上安装和配置 Docker Desktop 开始。你将学习如何运行你的第一个容器，理解容器化的基本概念及其优势。本系列将引导你完成构建你的第一个 Docker 镜像，深入了解如何创建高效且可复用的镜像。最后，你将探索如何将镜像发布到 Docker Hub，从而与更广泛的社区分享你的工作，并利用 Docker 强大的生态系统进行协作开发和部署。

## 你将学到的内容

- 设置 Docker Desktop
- 运行你的第一个容器
- 构建你的第一个镜像
- 在 Docker Hub 上发布你的镜像
