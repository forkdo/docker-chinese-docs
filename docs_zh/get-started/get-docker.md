Docker 是一个用于开发、交付和运行应用程序的开源平台。

Docker 让您可以将应用程序与基础设施分离开来，从而可以快速地交付软件。借助 Docker，您可以用管理应用程序的相同方式来管理您的基础设施。

通过利用 Docker 快速交付、测试和部署代码的方法论，您可以显著减少从编写代码到在生产环境中运行它的延迟。

您可以在多个平台上下载并安装 Docker。请参阅以下部分，并选择最适合您的安装方式。

> **Docker Desktop 使用条款**
>
> 大型企业（员工超过 250 人或年收入超过 1000 万美元）对 Docker Desktop 的商业使用需要 [付费订阅](https://www.docker.com/pricing/)。
