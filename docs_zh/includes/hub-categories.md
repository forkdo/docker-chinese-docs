这些类别包括：

- **API 管理**：用于创建、发布、分析和保护 API 的工具。
- **内容管理系统**：通过模板、流程和标准格式创建和管理数字内容的软件应用程序。
- **数据科学**：支持分析数据并生成可操作洞察的工具和软件。
- **开发工具**：协助开发人员创建、调试、维护和支持应用程序和系统的软件和实用工具。
- **数据库与存储**：用于存储、检索和管理数据的系统。
- **语言与框架**：编程语言运行时和框架。
- **集成与交付**：用于持续集成（CI）和持续交付（CD）的工具。
- **物联网**：支持物联网（IoT）应用的工具。
- **机器学习与人工智能**：针对人工智能和机器学习项目优化的工具和框架，例如预安装的数据分析、模型训练和部署所需的库和框架。
- **消息队列**：针对可靠、可扩展和高效的消息处理而优化的消息队列系统。
- **监控与可观测性**：通过指标、日志和追踪来跟踪软件和系统性能的工具，以及用于探索系统状态和诊断问题的可观测性工具。
- **网络**：支持数据交换以及连接计算机和其他设备以共享资源的仓库。
- **操作系统**：管理计算机上所有其他程序的软件，作为用户与计算机硬件之间的中介，同时监督应用程序和系统资源。
- **安全**：保护计算机系统或网络免受硬件、软件、电子数据被盗、未经授权访问或损坏，以及防止服务中断的工具。
- **Web 服务器**：为用户或其他系统提供网页、HTML 文件和其他资源的软件。
- **网络分析**：收集、测量、分析和报告网络数据以及网站访问者参与度的工具。
